Natours

"Advanced CSS and Sass" - by Jonas Schmedtmann

Introduksjon

Natours er en fiktiv nettside laget og designet av kursforfatteren Jonas Schmedtmann. Nettsiden er laget i forbindelse med kurset "Advanced CSS and Sass".

Det er gitt tillatetlse av kursforfatteren om å bruke denne nettsiden for både personlig og kommersiell bruk, men ikke å hevde det som sitt eget design.

Hva jeg har lært

  • Moderne CSS Teknikker
  • Hvordan CSS fungerer bak scenen
  • Bruke Sass i større prosjokter
  • Oppbygning av Sass filer, håndtere Media Queries, Globale variabler, osv.
  • Responsive Bilder i HTML og CSS for raskere nettsider
  • NPM bygge prosesser
  • Avansert CSS Animasjoner med @keyframes, animations og transition
  • Komponentbasert design, BEM, skrive gjenbrukbar kode
  • Avansert responsiv design: Media Queries, mobil-først vs desktop-først, em vs rem-enheter, osv.
  • Bygge video bakgrunnseffekter
